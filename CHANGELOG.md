# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-06-01

### Added
- Initial release of TRIGRS-PyWrap
- Python wrapper for the TRIGRS landslide hazard model
- Automated TRIGRS input file generation
- DEM preprocessing and hillshade generation
- Slope calculation from DEM
- Flow direction analysis using pysheds
- Geological data rasterization
- Catani soil thickness model
- Hazard assessment mode with uncertainty quantification
- Batch processing for parameter sensitivity analysis
- Export to ASCII format for TRIGRS compatibility
